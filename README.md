# How to run

## 1. Clone

    $ git clone https://github.com/devgmz/ARm.git

## 2. Install dependencies
    $ npm install 
## 3. Run
    npm start 

## .env
> API : [TMDB](https://www.themoviedb.org/)
* REACT_APP_API_KEY
* REACT_APP_API_URL
* REACT_APP_IMAGE_BASE_URL
